# Simple Virtualbox Kubernetes Vanila Lab

 * Based on article: https://medium.com/@mojabi.rafi/create-a-kubernetes-cluster-using-virtualbox-and-without-vagrant-90a14d791617

## Tools
 - Virtualbox
 - Vagrant

## Vagrant Use

 * Start up Environment
  - Add new image box
   'vagrant box add ...'
  - Bring up boxes
   'vagrant up'

 * Destroy Environment
  - Destroy machine
   'vagrant destroy'
  - Remove boxes
   'vagrant box remove ...'
